为此，我编写了一个傻瓜式一键修复脚本。作为玩家或是零基础无需纠结复杂的安装目录。

该脚本的原理是自动将 Steam 安装目录下的 steamwebrtc64.dll 文件复制到游戏对应的安装目录中，即可完成修复。

多合一版：

双击运行后，选择想要修复的游戏。脚本会自动完成剩下的工作。

现已支持：深岩银河、鼠疫2、雨中冒险2、遥遥西土、街霸6
